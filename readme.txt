--------feature.pyについて----------------
feature.pyはライブラリcaffeを用いて画像の特徴ベクトルを抽出する。
一度に数枚(設定してあるバッチ数)の画像を読み込み結果を出力するが、その中間層の値を抽出すると他の画像に影響されているかのような値が得られたため(プログラムの書き間違いの可能性もある)画像一枚一枚読み込むように書いてあります。
DATA_ROOTで指定したフォルダの中身は画像ファイルだけ、もしくはフォルダだけである必要があるので、画像とフォルダが混同してても画像のパスを取得できるように改良してください

~必要なこと~
・caffeの導入（これが一番難しい）
・画像が入っているフォルダのパスの設定

~使い方~
python feature.py と入力するだけ
------------------------------------------

--------tsneについて-----------------------
特徴ベクトルを２次元座標にしている。
基本的には　https://github.com/lvdmaaten/bhtsne　参照
入力の部分を少し書きなおしてます。
入力の特徴ベクトルをpcaで50次元にするように書きなおしてます。
perplexityの値は,画像数に応じて手作業で適切な値を探す必要がある。

~必要なこと~
・https://github.com/lvdmaaten/bhtsne にある通りにインストール
・perplexityの値の適切な値の探索

~使い方~
./bhtsne.py -i input.txt -o output.txt -p 195
このように入力する。
-i で入力テキスト(feature.pyの出力である特徴ベクトルのテキスト)、　-o で出力テキスト名, -p にてperplexityの値を入力。
------------------------------------------

-----------labels.pyについて--------------------------
ネットワークの構成とコミュニティ抽出、そしてt-sneで取得した各画像に対応する座標を用いてプロットする。
コミュニティごとに色が振り分けられるようになっている。
matplotlibにイベントを追加しました。具体的には、プロットされた点をクリックするとその点がハイライトされ、その点に対応する画像が別ウィンドウで表示されるようになっています。注意すべきなのが、強引にプログラムを書いたため,画像一枚表示したらその画像のウィンドウを閉じないと次のイベントが起きない仕様になっています。
thresholdという値はエッジ数の比率をthre関数に与えれば自動で得ることができる。

~必要なこと~
・いろんなライブラリを扱っているので各ライブラリのインストール

~使い方~
python labels.py　と入力するだけ
